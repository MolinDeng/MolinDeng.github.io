How to build this blog using Hexo and GitHub Pages

Posted on 2018-10-01 | Edited on 2018-10-03 | In Others | Views:
Symbols count in article: 5.1k | Reading time ≈ 5 mins.

Tools

  • Markdown：文章编写，内置层级、列表、超链接、图片插入、代码、数学公式等语法支持
  • Hexo：博客构建，快速搭建生成预览、发布
  • GitHub Pages：使用GitHub Pages托管，免费

Environment

  • Node.js

  • Hexo

    • 常用命令见Cheating Sheet

  • Git

Using NexT theme

  • 下载主题

  • 启用主题

    编辑_config.yml

  • 设置语言

    编辑_config.yml

Deploy GitHub Pages

  • 在自己的GitHub账号下创建username.github.io的repo

  • 设置SSH，详情参考廖雪峰的GIt教程

  • 安装git-deploy插件

  • 关联仓库：编辑_config.yml

  • 部署

Binding your Domain name

  • 购买域名

  • 等待实名认证

  • 解析域名

    配置如下2条

    • 记录类型A主机记录@记录值ip-addr
    • 记录类型CANME主机记录www记录值username.github.io

  • 在hexo-site/source/中新建CNAME（无后缀名）

  • 等待DNS生效

Hexo Configuration

  • 头像：在\hexo-site\themes\next\_config.yml中设置avatar: <avatar-url>

    • 可以放在\hexo-site\themes\next\source\images中直接引用url: /images/avatar.png

  • 添加tag页面

    • 编辑source/tags/index.md

    • 编辑themes/next/_config.yml

  • 新建categories页面，同上

  • 新建about页面，同上，编辑这个md文件就可以自我介绍了

  • 首页文章显示摘要<!--more-->

  • 显示文章更新时间、创建时间以及其他总结信息

    • 编辑themes/next/_config.yml

  • 显示文章统计信息

    • 编辑themes/next/_config.yml注意注释中的依赖

  • 代码块复制按钮、高亮

    • 编辑themes/next/_config.yml

  • 打赏信息

    • 编辑themes/next/_config.yml

  • 添加 Creative Commons 署名协议

    • 编辑themes/next/_config.yml

  • 字体设置在themes/next/_config.yml中的font字段

    • global全局字体
    • headings标头字体，就是h1,h2,h3…的字体
    • posts文章字体
    • logo左上角logo的字体Morning’s blog
    • codes代码快字体

  • 评论区

    • 注册livere账号

    • 生成livere_uid：在Code Managing中复制如下代码到配置文件中themes/next/_config.yml

  • 本地关键词搜索

    • 安装hexo-generator-searchdb 插件

    • 编辑_config.yml

    • 编辑themes/next/_config.yml

  • Fancybox

    • For install 3.x: https://github.com/theme-next/theme-next-fancybox3

    • 编辑themes/next/_config.yml

  • 右侧社交栏

    • 编辑themes/next/_config.yml查找social link字段

  • 腾讯公益404

    新建<hexo-site>/source/404.html

  • 站点生成时间

    • 编辑themes/next/_config.yml

  • 添加sitemap插件

    • 安装 hexo-generator-sitemap 插件

    • 编辑 _config.yml

  • 添加蜘蛛协议

    • 创建<hexo-site>/source/robots.txt

  • 设置RSS

    • 安装hexo-generator-feed 插件

    • 编辑 _config.yml
